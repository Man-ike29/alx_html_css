# CSS Basic Project

This project demonstrates basic CSS styling applied to HTML pages. It includes:

- `index.html` - Main portfolio page
- `tweets.html` - Tweets showcase page
- `base.css` - Base CSS styles
- `styles.css` - Custom styles

## Requirements

- W3C compliant HTML and CSS
- No external libraries or frameworks
- Built with pure HTML, CSS, and JavaScript only

## Files Structure

```
css_basic/
├── README.md
├── index.html
├── tweets.html
├── base.css
└── styles.css
```

## Usage

Open `index.html` in a web browser to view the portfolio page.
