# CSS Advanced Project

A modern, responsive website built with pure HTML, CSS, and JavaScript, focusing on advanced CSS techniques and styling.

## 📋 Project Overview

This project demonstrates advanced CSS concepts and techniques while creating a visually appealing, fully responsive website. The design is based on a Figma mockup and implements modern web development best practices.

## 🎯 Objectives

- Create a pixel-perfect implementation of the provided Figma design
- Utilize advanced CSS features including:
  - Flexbox and CSS Grid layouts
  - CSS animations and transitions
  - Custom properties (CSS variables)
  - Responsive design principles
  - Modern CSS selectors and pseudo-classes
- Ensure W3C compliance for both HTML and CSS
- Implement semantic HTML structure
- Optimize for performance and accessibility

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Advanced styling and layout techniques
- **JavaScript**: Interactive functionality (vanilla JS only)
- **No external libraries or frameworks**

## 🔧 Setup and Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd css_advanced
   ```

2. **Open in browser**
   - Simply open `index.html` in your preferred web browser
   - Or use a local server for development:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (if available)
   npx serve .
   ```

3. **Development**
   - Edit HTML in `index.html`
   - Modify styles in `styles/style.css`
   - Add JavaScript functionality in `scripts/main.js`
